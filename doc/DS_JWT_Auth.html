check_token() → {promise}

This is the key method for the object. It should be called before any API call to DocuSign. It checks that the existing access token can be used. If the existing token is expired or doesn't exist, then a new token will be obtained from DocuSign by using the JWT flow.
SIDE EFFECT: Sets the access token that the SDK will use
